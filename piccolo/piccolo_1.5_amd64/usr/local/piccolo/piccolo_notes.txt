Piccolo Notes
=============
Piccolo is a very minimal text editor intended for
applications requiring a simple display of monospaced
text or very simple editing.
There are neither tabs nor multiple documents.

Piccolo Features:

    will open file from command line
    remembers last file by default
    remembers position and shape on desktop
    simple search
    common keyboard editing shortcuts

Ctrl-q      to quit
Ctrl-s      to save
Shift-Ctrl-s to save-as
F2          to search
F3          for search next
Ctrl-o      to open a file
Ctrl-n      for new file
"u"         launch selected URL
